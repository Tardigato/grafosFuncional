Help

Si tienes dudas de como funciona la página aquí te lo explicamos.

En la página principal podrás visualizar dos botones uno te lleva a la sección de contactos donde verás la información relevante para contactarte con nosotras, y en esta página podrás ver como funcionan los algoritmos de ordenamiento (Sorts), en este apartado lo primero que visualizarás es que a tu derecha cuentas con una serie de botones y en el medio de tu pantalla un lienzo, que es donde ocurrirá la magia.

Cuando presiones el boton de Randomizar este creará una serie de números aleatorios, pero también puedes ingresar tu propia lista de números para ordenar. Si quieres ingresar una lista de números debe seguir el siguiente formato:

78,79,90,98,986,58,578,79,55,689
Esta es una lista de ejemplo, vos puedes poner los números que quieras, aunque te recomiendo para una mejor visualización que sean números altos dado que usa esa cantidad para la altura de las barras, lo único es que los números que ingreses deben estar separados por comas.

Si eliges la opción de ingresar tu propia lista de numeros no es necesario que modifiques la cantidad de elementos, dado que el gráfico se adaptará a la cantidad de números que ingresaste. Pero si por otro lado deciste usar la opción de Randomizar puedes elegir la cantidad de elementos a mostrar usando la barra de Elementos, esta es una barra que de puede deslizar de izquierda a derecha cuenta con un mínimo de 10 elementos y un máximo de 200 elementos. Esta barra está pensada para que sea más fácil que elijas cuantos elementos te gustaría ordenar.

¿Estás listo para la mágia? Solo falta que Selecciones el algoritmo a utilizar, cuando le des click te aparecerá una lista donde podrás elegir entre los distintos algortimos de ordenamiento como ser:
  • Shell Sort
  • Insertion Sort
  • Selection Sort
  • Merge Sort

Si ya te sientes cómo con tu lista para ordenar, lo único que debes hacer es darle click al botón de Sort el cual se encargará de que se ordenen tus elementos.
Te preguntarás ¿Y para qué es ese rectángulo blanco? sirve para que después de hacer el ordenamiento puedas visualizar cuanto tiempo demoró en ejecutarse el algoritmo en milisegundos.
